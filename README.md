# Structura: DSA Simulator

Structura is a Python-based Data Structures and Algorithms (DSA) simulator designed to help students visualize and understand fundamental DSA concepts through interactive graphical simulations.

This project was developed as a final requirement for the Data Structures and Algorithms course.

---

## üß† Implemented Data Structures & Algorithms

- **Stack**  
  - Implemented through a Parking Lot simulation  
  - Demonstrates LIFO (Last In, First Out) behavior
  
- **Queue**  
  - Implemented through a Parking Lot simulation  
  - Demonstrates FIFO (First In, First Out) behavior

- **Binary Tree**  
  - Visual representation of tree nodes and traversal

- **Binary Search Tree (BST)**  
  - Demonstrates ordered insertion and searching

- **Recursion**  
  - Tower of Hanoi simulation with animated disk movement
